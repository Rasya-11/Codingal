Hobbies

Coding

Coding is the process of writing instructions for computers to perform tasks using programming languages, such as Python or Java. Many people often do it in their free time to learn and explore new things!

Video games

Video games are electronic games played using an input device like a controller or keyboard to interact with a user interface. Many find this source of entertainment interesting and addicting

Reading

Reading is a popular and beneficial hobby that can be done alone or with others, offering relaxation and intellectual growth. It provides benefits such as improved vocabulary, better communication skills, and a greater understanding of the world.

Drawing

Drawing is a versatile hobby that can be done with minimal cost, offering relaxation, a creative outlet, and mental health benefits like stress reduction and improved focus.
